# Гра "Вгадай слово"

## Опис
Консольна гра, де комп'ютер загадує слово, а гравець намагається його відгадати, вводячи букви по одній.

## Правила гри
1. Комп'ютер обирає випадкове слово зі списку.
2. Гравець бачить кількість літер у слові (відображаються як _).
3. Гравець вводить одну букву.
4. Якщо буква є у слові, вона відкривається на відповідних місцях.
5. Якщо букви немає, гравець отримує повідомлення про помилку.
6. Гра завершується, коли все слово відгадане.

## Технічні вимоги
* Мова: Java
* Введення: Через консоль (Scanner).
* Логіка: Використання масивів (char[]) та циклів для перевірки букв.
* Дизайн: Модульна структура всередині класу (розділення логіки вибору слова та ігрового циклу).

## Як запустити
1. Відкрити проєкт в IntelliJ IDEA.
2. Запустити клас Game.java в пакеті GuessTheWord.
3. Слідувати інструкціям у консолі.
